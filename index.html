Programa LTCHF

Este es un programa creado por Jerónimo Mejía para hacer registros de lectura. Permite anotar libros, autores, fechas, opiniones y muchas otras cosas relacionadas con tus lecturas.
Recuerda que la contraseña se cambia cada dia.
Acceso correcto. Bienvenido al Programa Jeronimo Mejia.

Descripción: Este programa creado por Jerónimo Mejía sirve para registrar lecturas de una forma sencilla. Permite guardar información de libros, capítulos o páginas, copiar y organizar textos, y acceder fácilmente a herramientas externas para procesarlos o analizarlos. Está diseñado para ayudarte a tener tus lecturas bien ordenadas y listas para trabajar con ellas.

Guia para utilizar el Programa

A continuación, para manejar el programa necesitas estos pasos:

  1. Toma una captura de cada página Debes una foto de todas las páginas, una por una.
    Ejemplo: si tu primera página es la 54, tómale una foto. Luego selecciona el texto que aparece y cópialo. Para hacerlo, toca el símbolo de la cámara, que está al lado del símbolo de voz.

    Aquí está el enlace para ir: Tocar
  2. Guarda y organiza el texto Una vez copies el texto, guárdalo. Repite este proceso con todas las páginas o capítulos que necesites.
    Ejemplo: si estás trabajando con los capítulos 11 al 13, guarda todos esos textos. Luego entra a la otra aplicación (si no tienes cuenta, créala) y pega el texto completo.
    Puedes pedirle al programa lo que quieras hacer con cada texto. Ejemplo: Capítulo 24: texto…”, “Capítulo 25: texto…”, y así sucesivamente. En resumen, solo copia y pega el contenido.

    Aquí está el enlace para ir: Tocar
  3. Repite el proceso con cada capítulo o conjunto de páginas que quieras registrar. Así podrás tener toda tu lectura organizada y lista para trabajar con ella.
